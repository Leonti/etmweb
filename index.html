ETMweb - Employee Time Management - web application using ETM library

By Leont Bielski

LaborTM - website using this application

Labor Time Management is a web service for employers that tracks time employees spend on their tasks by providing easy to use time tracking interface. Who is it for?

Organizations where job pays are hourly based such as hotels, restaurants, stores. Key features

Employees log in and log off through simple web interface with personal codes
Each employee can have multiple jobs assigned to him
Overtimes are calculated with ability to adjust to organization overtime policy
